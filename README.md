# SDE-BBHost-Run-Checker

## to 데이터 분
* 프로그램 사용 중 에러나 개선 사항 있으면 이슈에 올려주세요. 

## to 박문수 과장님
* 이메일 수신자를 추가 또는 수정하고 싶으시면 administrators.txt를 수정하시면 됩니다.
* 수정시 주의 사항: 다음 줄로 넘어가지 말고, 이메일 주소를 한 줄에 콤마(,)로 구분하여 입력해주시기 바랍니다.
